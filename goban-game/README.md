# 五子棋游戏

一个基于 PHP 和原生 JavaScript 开发的网页版五子棋游戏，支持用户注册登录、AI 对战、积分排行榜等功能。

## 功能特性

- **用户系统**：支持用户注册、登录、游客模式
- **AI 对战**：8 秒等待后自动匹配 AI 对手，AI 具备基础攻防策略
- **积分系统**：根据游戏结果增减积分，激励玩家获胜
- **排行榜**：实时展示前 10 名玩家的积分、对局数和胜率
- **游戏操作**：支持悔棋、求和、认输等常用操作
- **响应式设计**：适配桌面端和移动端

## 项目结构

```
goban-game/
├── index.php      # 登录/注册页面
├── game.php       # 游戏主页面
├── rank.php       # 排行榜页面
├── functions.php  # 后端接口处理
├── style.css      # 全局样式
└── users.db       # 用户数据文件（JSON 格式）
```

## 技术栈

- **后端**：PHP（使用文件型 JSON 数据库，无需 MySQL）
- **前端**：HTML5 + CSS3 + 原生 JavaScript
- **数据存储**：JSON 文件（users.db）

## 安装与运行

### 环境要求

- PHP 5.6 及以上版本
- 任意 Web 服务器（Apache / Nginx / phpStudy 等）

### 部署步骤

1. 将项目文件放置到 Web 服务器根目录下
2. 确保 `users.db` 文件所在目录具有读写权限
3. 在浏览器中访问 `index.php` 即可开始游戏

### 本地快速启动

使用 PHP 内置服务器：

```bash
cd goban-game
php -S localhost:8000
```

然后在浏览器中打开 `http://localhost:8000/index.php`。

## 游戏规则

### 基本玩法

- 棋盘大小：15×15
- 黑棋先行，双方轮流落子
- 率先在横、竖、斜任一方向连成 5 子者获胜

### 积分规则

| 结果 | 积分变化 |
|------|----------|
| 获胜 | +7 分（基础分 2 分 + 获胜奖励 5 分） |
| 和棋 | 0 分（基础分 2 分 - 求和扣 2 分） |
| 认输 | +2 分（仅基础分） |
| 悔棋 | -1 分（AI 对战限悔一次） |

### 操作说明

- **悔棋**：撤销上一步棋，AI 对战中仅限使用一次
- **求和**：向对方提出和棋申请，AI 对战直接和棋
- **认输**：主动放弃本局，对方获胜

## 页面说明

### 登录页（index.php）

- 用户可通过用户名和密码登录
- 新用户可点击"立即注册"创建账号
- 支持游客模式登录（不记录积分）

### 排行榜页（rank.php）

- 展示积分前 10 的玩家信息
- 当前用户行会高亮显示
- 每 5 秒自动刷新排行榜数据
- 可创建房间或加入房间开始游戏

### 游戏页（game.php）

- 8 秒倒计时后自动开始 AI 对战
- 可选择黑棋或白棋（默认黑棋先行）
- 实时显示双方玩家信息和当前回合
- 游戏结束后可再来一局或返回排行榜

## 接口说明

所有接口均通过 POST 请求 `functions.php`，参数 `action` 指定操作类型。

| action | 说明 | 参数 |
|--------|------|------|
| login | 用户登录 | username, password |
| register | 用户注册 | username, password |
| getLeaderboard | 获取排行榜 | 无 |
| getCurrentUser | 获取当前用户 | 无 |
| updateScore | 更新用户积分 | username, scoreChange, gameResult |
| logout | 退出登录 | 无 |

## 注意事项

- 本项目使用 JSON 文件存储数据，适用于小型或演示用途，高并发场景建议改用数据库
- 密码使用 PHP `password_hash()` 函数加密存储
- 当前版本仅支持人机对战，双人联机功能预留了接口但未完全实现
