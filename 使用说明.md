# 民生诉求反馈平台使用说明

## 最简单启动方式

双击根目录里的：

```text
启动正式服务.bat
```

启动后浏览器会自动打开：

```text
http://127.0.0.1:3000
```

如果手机和电脑在同一个 Wi-Fi，需要用电脑的局域网 IP 访问，例如：

```text
http://192.168.111.212:3000
```

## 常用脚本

- `启动正式服务.bat`：正式运行，给演示、内网访问、部署验证使用。
- `开发调试.bat`：开发模式，前端热更新，适合继续修改页面。
- `重新构建.bat`：修改代码后重新生成生产文件。
- `项目检查.bat`：执行类型检查、构建检查和接口检查。

## 默认账号

- 市级账号：`city / 123456`
- 东城区县级账号：`county-east / 123456`
- 西城区县级账号：`county-west / 123456`

## 重要目录

- `data/complaints.sqlite`：数据库文件，不能随便删除。
- `uploads/`：用户上传的附件，不能随便删除。
- `docs/`：技术文档和部署教程。
- `client/`：前端源码。
- `server/`：后端源码。
- `dist/`：构建后的正式运行文件。
